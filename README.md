A modern, high-performance portfolio built with **React 19**, **Vite**, and **Tailwind CSS v4**. This project showcases a collection of full-stack applications, AI integrations, and enterprise-grade systems.

## ✨ Features

- **🚀 Performance-First**: Built with Vite for lightning-fast HMR and optimized production builds.
- **🎨 Modern Aesthetics**: Clean, minimalist design with a dark/light mode toggle.
- **🎭 Smooth Animations**: Powered by **Framer Motion** for a premium user experience.
- **📱 Fully Responsive**: Seamlessly adapts to mobile, tablet, and desktop screens.
- **📩 Contact Integration**: Functional contact form powered by **EmailJS**.
- **📂 Dynamic Project Pages**: Detailed views for each project with image galleries.

## 🚀 Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Minel03/React-Portfolio.git
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env.local` file in the root directory and add your EmailJS credentials:

   ```env
   VITE_EMAILJS_SERVICE_ID=your_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_template_id
   VITE_EMAILJS_PUBLIC_KEY=your_public_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
